User Ratings Plugin

Allow users to rate movies, TV shows, episodes, and music. Ratings and reviews are visible to all users on the server via the web interface.
Number of items to display in each "Recently Rated" section (Movies, Shows, Episodes). Default is 10.

Data Management

Export your ratings to a JSON file or import them from a backup.

Danger Zone

Permanently delete all user ratings from the database. This action cannot be undone.
